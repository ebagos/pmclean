```
複数のディレクトリに保存されているファイルに関して、重複ファイルを削除するプログラムを作成する
・ファイルが同一であるか否かはハッシュ値が合致するか否かで決定する
・ハッシュ値が一致したファイルの中で最新のファイルを残し、他のファイルは削除する
・ハッシュロジックはmd5、sha1、sha256、sha512のいずれかとする
・ディレクトリの指定、およびハッシュロジックはJSON形式で記述された設定ファイルで与える
・設定ファイルはコマンドライン引数で与えることとする
・設定ファイルの指定が省略された場合はカレントディレクトリのconfig.jsonを用いるものとする
・以下は設定ファイルの例である
{
    "dirs_path": [
        "D:\\home\\Downloads\\ppv",
        "E:\\home\\Downloads\\ppv"],
    "hash_algo": "md5"
}
・各ファイルのハッシュ値はそのディレクトリにJSON形式で保存し、ファイル名はresults.jsonを用いるものとする
・プログラムの高速化のために、プログラム起動時にresults.jsonを読み込み、ハッシュ値の計算を省略すること
・ただし、ファイルの変更、追加、削除があった場合は、ハッシュ値の計算を行うこと
・処理の進行状況がわかるように、どのディレクトリで全体のファイルのうちどれぐらいのファイルを処理したかがわかるように途中経過を表示すること
Pythonで記述せよ
```
